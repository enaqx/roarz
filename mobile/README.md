#### Roarz
Real-time group-chat mobile app

##### Building:
 * Install [git](http://git-scm.com/), [node.js](http://nodejs.org/) and 
  [Ionic](http://ionicframework.com/getting-started/)
 * `$ git clone https://github.com/enaqx/roarz`
 * `$ cd roarz/mobile`
 * `$ npm install`
 * `$ ionic setup sass`
 * `$ ionic serve`
